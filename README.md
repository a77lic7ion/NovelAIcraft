## ğŸš€ Getting Started

### Prerequisites
*   A modern web browser.
*   (Optional) A **Google Gemini API Key** for cloud-based intelligence.
*   (Optional) A running **Ollama** instance for local-first intelligence.

### Installation
1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/NovelAIcraft.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Launch the studio:
    ```bash
    npm start
    ```
